# MAESTRO Frontend

React + TypeScript + Vite dashboard for the MAESTRO agentic lab orchestration platform.

## Stack

- **React 19** + **TypeScript**
- **Vite** (build) + **Rolldown** (bundler)
- **TailwindCSS v4** (styling)
- **Zustand** (state management)
- **React Router v7** (routing)
- **Recharts** (charts)
- **React Flow (@xyflow/react)** (workflow DAG + digital twin)
- **react-markdown** (LLM response rendering)
- **Lucide React** (icons)

## Development

```bash
npm install
npm run dev        # starts on http://localhost:3000
